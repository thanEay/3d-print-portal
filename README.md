# 3D Print Portal 

Made during Hack Club Stardance, it allows students at a school to upload models that they want 3D printed, and an administrator or teacher can start the prints. 

## Project Description

- Students can upload 3D models to a website that is exposed to the network
- Two main pages, one for an admin/teacher and one for any students that want to upload 3D models
- Uses a database/spreadsheet (tbd) to hold student names, emails, timestamp of uploads, print colors, filament types, some basic print settings like infill, any notes about the print, and the path to a folder with the models 
    - All of these other than timestamp and file path are input manually by the student

## Purpose

This project is meant to solve a real problem that I have. I am a student, but I have nearly sole access to my school's Bambu Lab H2D, by far the nicest (and most expensive) 3D printer that my high school has ever had. Outside of a few technically minded individuals, most people are unaware of its existence, and those who are don't have an easy way to use it. The printer is too expensive to let technically ignorant students start prints all on their own and risk damaging it, but everyone in the school should be able to use it. 

By creating this project, I want to give students in my school the ability to use the amazing piece of technology that is the Bambu Lab H2D without risking students damaging it out of ignorance.
